# Abstract VIII: Relinquishment

## Prerequisite Physics

This abstract makes no physics claims. It is an interpretive reading of a published essay (Bill Joy, "Why the Future Doesn't Need Us," Wired, April 2000).

## Abstract

April 2000: prominent computer scientist publishes essay in Wired expressing existential alarm about self-replicating technologies. Three target technologies conspicuously exclude quantum computing. The essay reads differently if relinquishment is already too late.
